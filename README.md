# Zirconium
odin addon for latest versions\
\
download from the latest action in:\
https://github.com/poisonous888/Zirconium/actions \
you will also need odin from:\
https://modrinth.com/mod/odin \
or https://github.com/odtheking/Odin \
\
other suggested mods:\
https://modrinth.com/mod/rrls \
https://modrinth.com/mod/no-telemetry \
https://modrinth.com/mod/no-chat-restrictions \
\
feature list (idk how to do the fancy arrows here):\
\
Held Item Render\
|- held item transform  - move held item (think odin animations)\
|- swing transform - change how items move around when swung (things like no swing and custom swing duration)\
|- hand and swap\
-- |- no swap (item instantly changes when swapping hotbar slots)\
-- |- disable hand\
-- |- sync hand with item position\
|- presets - feel free to suggest new ones, I’m always looking to flesh them out more\
\
Misc Features\
|- no loading screen (like 189 patcher)\
|- update notif - sends message on first world load if your not on the most recent version\
|- close sign on enter - closes sign GUI's when you press the enter key (189 thing)\
|- no recipe book\
|- no potion effects - removes the potion effect hud in the top left and in your inventory\
|- no server packs - prevents server packs from being downloaded and applied\
|- no required packs - allows you to move ANY pack in the resource pack menu (disabling the vanilla pack WILL crash the game, ive tried it)\
\
Drop Utils\
|- drop protection - standard item protection\
-- |- settings for protecting: recombed, starred, museum donated, all\
-- |- bindable keys that mark items protected by skyblock ID and UUID\
-- |- note that i havent gotten around to doing special tooltips for them so theres a dedicated setting to highlight protected items
|- drop stack key\
\
Visuals\
|- HUGE props to animatium for the base mixins\
|- diagonal backwards walk - readds the ~45 degree rotation when walking backwards\
|- custom cape physics - lets you change how the cape clamps its movement (and make it more goofy like 189)\
\
Custom Commands\
|- custom aliases and keybinds that preform commands\
|- look into /alias and /keybind for more info\
\
Chat Utils\
|- custom chat rules that can hide messages and/or send an alert with a title and sfx\
|- look into /chatrule or just /rule for more info\
|- preset chat rules for common messages - pickaxe ability, blocks in the way, kill combo, ability damage, item on cooldown\
|- (future) chat copy, compact chat, 
\
Static Waypoints\
|- custom island specific waypoints\
|- look into /staticwaypoints or just /sw for more info\
\
GUI Highlight\
|- highlight active pet\
|- highlight completed commissions\
|- anvil helper (enchanted book combine helper like skytils or skyblocker)\
\
(future) Zoom\
|- keybind to zoom FOV (like optifine)\
\
Hide Armor\
|- hides specified pieces of player worn armor (like 189 odin)\
\
Garden\
|- pest equipment swap alert\
|- pest cooldown hud\
|- (future) pest glow & tracer\
\
Mouse Lock\
|- prevents mouse movement from rotating player\
\
CPS Display\
|- readds the CPS display from 189 odin (some1 requested)\
\
DVD\
|- readds the bouncing DVD hud from 189 odin (some1 requested)\
\
Lag\
|- decreases memory usage\
